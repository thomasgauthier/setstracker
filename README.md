# Sets Tracker

A simple React app for tracking gym sets and reps.

## Features

- Track exercises with sets and reps
- Add new exercises dynamically
- View workout history grouped by date
- Local storage persistence
- Clean, responsive UI with Tailwind CSS

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18
- Tailwind CSS
- esbuild
- Local Storage for data persistence
